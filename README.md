# Customer Churn Analysis

## Project Overview

This project analyzes customer subscription data to identify factors that contribute to customer churn. The analysis focuses on customer tenure, monthly charges, and engagement patterns to understand cancellation behavior and improve retention strategies.

## Objective

* Analyze churn patterns
* Identify high-risk customer segments
* Evaluate customer tenure and spending behavior
* Generate insights for retention improvement

## Technologies Used

* Python
* Pandas

## Files

* customer_data.csv
* task2.py
* churn_analysis_report.csv

## Features

* Missing value handling
* Churn percentage calculation
* Tenure analysis
* Monthly charges analysis
* Automated report generation

## Execution

```bash
pip install pandas
python task2.py
```

## Output

The program generates:

* Churn statistics
* Customer behavior insights
* churn_analysis_report.csv
